# Contributing to NYC Taxi Analytics Platform

Thank you for your interest in contributing! This is a portfolio project, but improvements and suggestions are welcome.

## How to Contribute

1. **Fork the Repository**
   ```bash
   git clone https://github.com/chanderbhanu096/nyc-taxi-ml-pipeline.git
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Your Changes**
   - Follow existing code style
   - Add comments for complex logic
   - Update documentation if needed

4. **Test Your Changes**
   ```bash
   python3 main.py  # Test the pipeline
   streamlit run src/analysis/dashboard.py  # Test the dashboard
   ```

5. **Commit and Push**
   ```bash
   git add .
   git commit -m "Add: your feature description"
   git push origin feature/your-feature-name
   ```

6. **Submit a Pull Request**

## Code Style

- Use Python type hints where possible
- Follow PEP 8 guidelines
- Keep functions focused and well-documented
- Use meaningful variable names

## Questions?

Feel free to open an issue for discussion!
